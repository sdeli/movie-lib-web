Jacpot Game

Description
  • This is a movie library, which I have created, to help me get hired... hopefully:)
  • It was actually requested by a company, which is situated on a beautiful island:)
  • All the data is scraped from this website: www.imdb.com/....
  • It has postgres database where the data - from the above mentioned website - lives and gets served, furthermore a nestjs backend and Angular frontend.
  • Some technologies I was using on the back and the front: typeorm, typeorm crud, nestjs crud request, rxjs, ngrx, lodash, material ui components etc..
  • When creating this, I was in kind of a hurry, so some things of course could have been done better, for example I have used 'px' as main unit, but I know in some cases I should have used 'rem', and at some points code may could be cleaner, but for now I think it should be fine.
  • For writing the html and css, I have used some BEM notation where it made sense.
  • Finally please enjoy the application, I am looking forward to hear about you:)
